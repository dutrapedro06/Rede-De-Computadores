# 🌐 Redes de Computadores

Repositório da disciplina de **Redes de Computadores**, contendo atividades práticas, laboratórios e trabalho final desenvolvidos ao longo do curso.

## 📚 Sobre a disciplina

A matéria aborda os conceitos fundamentais de redes, incluindo comunicação entre dispositivos, protocolos e simulações práticas.

## 🧠 Conteúdos abordados

- ✔️ Modelo OSI e TCP/IP  
- ✔️ Endereçamento IP  
- ✔️ Sub-redes (Subnetting)  
- ✔️ Protocolos de rede (HTTP, DNS, TCP, UDP)  
- ✔️ Roteamento  
- ✔️ Comunicação cliente-servidor  
- ✔️ Simulações de rede  

## 💻 Tecnologias utilizadas

- Python 🐍  
- Simuladores de rede (ex: Packet Tracer)  
- Conceitos teóricos aplicados na prática
